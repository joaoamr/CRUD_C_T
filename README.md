# CRUD_C_T
A CRUD exercise made in C# and Javascript using Visual Studio.

# DB Settings

## Connection
The database access was implemented using ADO.Net and tested with SQL Server Express 2005. To change the the connection directives modify the local global variable __connectionString__ in at the class __CRUD_C_T/App_Code/models/DbSettings.cs__.

## Tables
Create a new database CLINTEDB and run the script __CreateTables.sql__ for it.
